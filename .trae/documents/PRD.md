## 1. 产品概述

TimeForge 是一个面向全球开发者的中英双语在线时间戳转换工具站，核心解决 Unix 时间戳与人类可读日期之间快速互转、时区换算与相对时间计算的需求。
- 目标用户：后端/前端开发者、运维工程师、数据处理人员、跨境协作团队
- 产品价值：单一工具页精准命中高商业价值开发者搜索词，纯前端实现零服务器成本，通过 AdSense 实现长期被动广告收入

## 2. 核心功能

### 2.2 功能模块
1. **工具首页（/）**：实时时间戳显示、时间戳⇄日期互转、时区选择、相对时间计算、批量转换
2. **关于页（/about）**：产品定位、技术说明、联系方式
3. **隐私政策页（/privacy-policy）**：数据不收集声明、Cookie/广告说明、AdSense 合规条款
4. **404 页面**：友好的错误提示与回首页入口

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|-----------|-------------|---------------------|
| 工具首页 | 实时时钟面板 | 顶部显示当前 Unix 时间戳（秒/毫秒），每秒更新，一键复制 |
| 工具首页 | 时间戳→日期 | 输入时间戳自动解析为多时区可读日期，支持秒/毫秒自动识别 |
| 工具首页 | 日期→时间戳 | 日期时间选择器 + 时区下拉，输出对应 Unix 时间戳 |
| 工具首页 | 时区矩阵 | 同一时刻在 UTC/本地/纽约/伦敦/东京/上海等主流时区的对照表 |
| 工具首页 | 相对时间计算 | 加减秒/分/时/天/月，输出目标时刻与时间戳 |
| 工具首页 | 批量转换 | 文本框粘贴多行时间戳，一键全部转换为可读日期 |
| 工具首页 | 知识卡片 | Unix 时间戳定义、2038 问题、各语言获取时间戳代码片段 |

| 隐私政策页 | 政策条款 | 数据零收集声明、第三方广告 Cookie 说明、用户权利 |

## 3. 核心流程

用户进入站点 → 默认看到实时时钟与当前时间戳 → 在互转区输入/粘贴数据即时获得结果 → 复制结果到剪贴板 → 切换语言或查阅时区矩阵 → 完成需求离开（高停留率触发广告曝光）。

## 4. 用户界面设计

### 4.1 设计风格
- **设计方向**：精密仪器 / 终端美学 + 现代精致——深色仪表盘风，但排版克制、留白慷慨、层级清晰，体现开发者工具的可信感
- **主色**：深炭黑背景 `#0A0E1A` + 墨蓝次级面板 `#13182A`
- **强调色**：电光青柠 `#C6FF3D`（单一锐利点缀，用于关键数据/CTA/激活态）
- **次要文字**：冷灰 `#8A93A8`
- **按钮风格**：低圆角（4px）、扁平、激活态用强调色描边+微光，复制成功按钮变绿短暂反馈
- **字体方案**：
  - 数据/数字/代码：`JetBrains Mono`（等宽，体现技术精度）
  - 标题/正文：`Space Grotesk`——但为避免雷同改用 `Outfit`（几何无衬线，独特清爽）
  - 中文：`Noto Sans SC`
- **布局风格**：单栏垂直流，卡片分区，顶部固定导航含 Logo + 语言切换 + 主题切换
- **图标风格**：线性细描边 SVG，1.5px stroke，与字体气质一致

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI 元素 |
|-----------|-------------|-------------|
| 工具首页 | 实时时钟面板 | 顶部大号等宽数字时间戳，秒级跳动，微弱呼吸光晕，下方 UTC ISO 字符串小字 |
| 工具首页 | 互转卡片 | 左右双栏卡片（移动端堆叠），输入框带聚焦描边，输出区高亮强调色边框 |
| 工具首页 | 时区矩阵 | 表格行式布局，每行一个时区，左侧城市名右侧时间，hover 行高亮 |
| 工具首页 | 知识卡片 | 折叠式手风琴，展开后代码块带语法高亮色块底 |
| 关于页 | 介绍区 | 居中排版，左侧产品定位文案，右侧技术栈标签云 |
| 隐私政策页 | 条款区 | 长文阅读优化，左侧锚点目录，右侧正文，关键术语用强调色 |

### 4.3 响应式
- 桌端优先：1280px+ 双栏布局充分利用宽度
- 平板：768-1024px 单栏卡片堆叠，时区矩阵保留表格
- 移动端：<768px 全部单栏，时间戳数字字号自适应缩小，触摸友好的按钮尺寸（min 44px）
- 主题切换：默认深色，支持切换至浅色（米白底 `#F5F2EA` + 墨黑文字，保持强调色）

### 4.4 动效
- 页面加载：模块逐项淡入上移（staggered 80ms 延迟）
- 实时时戳：数字翻转微动效（不抢戏）
- 复制反馈：按钮态切换 + 短暂 toast 提示
- 语言切换：内容无刷新过渡，0.2s 淡入
- hover：时区矩阵行/按钮/卡片边框微光过渡
